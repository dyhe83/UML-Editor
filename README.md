# UML-Editor
[![java8](https://img.shields.io/badge/java-8.0-orange.svg?style=popout)](https://www.java.com)

一個簡單的 Java UML 編輯器

## How to use
```
$ git clone https://github.com/dyhe83/UML-Editor.git --depth 1
$ cd UML-Editor
$ gradle run
```
